# Vehicle-Damage-Classification-Using-CNN-for-Insurance-Claims
### Deep learning model automating vehicle damage classification for insurance claims
This project applies deep learning to automate vehicle damage classification from images, aimed at streamlining insurance claim workflows. A Convolutional Neural Network (CNN) was trained on a dataset of 7,200 images categorized into five damage types: scratches, flat tires, dents, broken windows, and damaged lamps.
